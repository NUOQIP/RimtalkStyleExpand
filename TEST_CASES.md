# RimTalk StyleExpand 测试用例

## 测试环境准备

1. RimWorld 1.5 或 1.6
2. 安装 RimTalk Mod
3. 安装 StyleExpand Mod
4. 配置向量 API（Ollama 或云端 API）
5. 准备测试文风文件

---
